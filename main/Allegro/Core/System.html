    System lifecycle, timing, and system info helpers for Allegro 5.

    Provides initialization, shutdown, timing utilities, and runtime information such as the library version, app/org name, and hardware.

    Version decoding #

    The packed version from getAllegroVersion encodes major/minor/revision/release:

    let v ← Allegro.getAllegroVersion
let major := v >>> 24
let minor := (v >>> 16) &&& 0xFF
let rev   := (v >>> 8) &&& 0xFF
let rel   := v &&& 0xFF
