    Allegro 5 File I/O bindings (file.h) #

    Full bindings for Allegro's virtual-file-system abstraction. Every *_f variant across all addons takes an AllegroFile handle.

    Seek origins #

    • seekSet (0) — seek from beginning
    • seekCur (1) — seek from current position
    • seekEnd (2) — seek from end
