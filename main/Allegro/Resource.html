    RAII-style resource wrappers for Allegro 5 #

    Each with* function creates (or loads) a resource, passes it to a callback, and guarantees destruction via try / finally — even if the callback throws.

    Example #

    Allegro.withDisplay 640 480 fun display => do
  Allegro.withEventQueue fun queue => do
    Allegro.withEvent fun event => do
      -- game loop --
      pure ()

    All wrappers are polymorphic in the return type (IO α), so they compose naturally with do-notation.

    def Allegro.withDisplay {α : Type} (w h : UInt32) (f : Display → IO α) :
    IO α

    Create a display of size w × h, run f, then destroy it.

    Equations
    • Allegro.withDisplay w h f = do let d ← Allegro.createDisplay w h tryFinally (f d) (Allegro.destroyDisplay d)
