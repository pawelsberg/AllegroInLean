    Image addon bindings #

    Provides bitmap loading/saving in various image formats (PNG, BMP, JPG, etc.). The image addon must be initialised before loading bitmaps from files. Drawing and lifecycle functions are in Allegro.Core.Bitmap.

    Typical usage #

    let _ ← Allegro.initImageAddon
let bmp ← Allegro.loadBitmap "sprite.png"
-- use drawBitmap etc. from Core.Bitmap --
Allegro.destroyBitmap bmp
